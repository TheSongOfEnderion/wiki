An armor gifted to [[Marcus Romulo]] by [[Kataratch]] himself after he bested him in single combat. It is forged by [[Kataratch]] using his body and soul, a powerful armor capable of protecting him against an [[Eldrasil]]. [[toc]] # Appearance The Kataratch Armor is a jet black armor made from the black ebony body of [[Kataratch]] himself. It possess the characteristics insect-like shell of a [[Reaver]]. # Ability The Kataratch Armor boast a wide number of abilities, all of which comes from [[Kataratch]] himself. * **Invulnerability** - Makes the user invulnerable to most damage, physical or magical. * **Hypermagic Capacity** - Gives the user a great amount of magicka. * **Hyperstamina** - Gives the user an almost everlasting stamina. * **Hyperspeed** - Givers the user speed faster than sound. * **Hyperstrength** - Gives the user strength greater than a mountain. * **Hyperthought** - Gives the user ability to think faster. * **Hyperfortify Magic** - Gives a powerful boost to every magic casted by the user. * **Space Rippling** - Lets the user ripple and destroy space itself. * **Gravity Orbs** - Lets the user create powerful gravity orbs. * **Weight Manipulation** - Lets the user manipulate the gravity of the armor and its surrounding at will. ** **Featherfalling** - Makes the user invulnerable from fall damage. * **Blink Teleportation** - Let's the user instantaneously teleport within a hundred meters radius. * **Dimension Transportation Immunity** - Gives the user resistance from being forcibly teleported. * **Elemental Immunity** - Makes the user invulnerable to all elemental damage. * **Soul Immunity** - Makes the user invulnerable to any types of soul damage. * **Chaos Immunity** - Makes the user invulnerable to all chaos damage. * **Flight** - Let's the user fly. * **Water Breathing** - Let's the user breathe underwater. * **Soul Magic** - Let's the user use soul magic. * **Muffled** - Makes the user hide their sound when sneaking. * **Quiet Casting** - The user can cast any magick without any sound. * **Fortified Magicka Regeneration** - The user's magicka regenerates at an incredibly fast rate, * **Technique Copying** - Let's the user copy any combat technique they witness. *
